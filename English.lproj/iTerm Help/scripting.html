 Scripting support:
    iTerm has basic Applescript support allowing one to write stand-alone scripts to launch the application and open multiple sessions from the addressbook into either new tabs or new windows. The syntax is as follows:

    launch session "session_name" in tab
    launch session "session_name" in terminal

    The first command opens the session named "session_name" in a new tab, while the second opens it in a new window. A session with that name has to exist in the address book. If a default session is desired, use "default session" as the session name. New tabs will be placed in the top active window. Here is a sample Applescript:

    tell application iTerm
    activate
    launch session "Root Shell" in tab
    launch session "default session" in terminal
    launch session "Fink list" in tab
    end tell

    These scripts can then be saved as stand-alone executable applications.
