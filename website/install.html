 System Requirements:
    Mac with Mac OS X 10.2 (Jaguar)
Install:
    Mount the image file and copy iTerm to anywhere you want. If you have an older version, be sure it's either over-written or completely gone (i.e. not in trash). OS X sometimes has weird behavior about that.
Tips:
  • Starting from 0.7.0, FreeMono font is bundled with iTerm. It is a monospace font with graphical characters that are required by many console programs.
  • Text alignment in iTerm depends on the font you selected. Currently, to have good alignment, you have a limited choice of fonts, especially when you need to use graphical characters. Bold is also only supported when the chosen font has a bold trait available that does not change the font height.
  • Starting from version 0.5.5, iTerm uses a different approach to support all encodings. Thus the old encoding settings are lost and they have to be manually reset.
  • If you are originally running CTerminal and wish to keep your old bookmarks, you need to rename your "CTerminal Bookmarks" file as "iTerm Bookmarks".
  • To get precise character position matching, please choose Osaka-regular mono font. This is the only font that I find to be really even spaced.
  • The bookmarks is stored in ~/Library/Application SupportIt's not encrypted, so be careful of your sensitive information
  • In the command field in Quick Run and Bookmarks windows, you can type in arguments for the command you wish to run. For example, if you want to run bash with existing environment settings, use bash --login
  • To be able to transfer CJK characters in a telnet session, you might want to use -8 option. eg: telnet -8 fsbbs.org 3000
