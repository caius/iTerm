How to install localization packages
Download the localization package you need and unzip it. Quit iTerm if you are running it. Open the iTerm.app package by CTRL-clicking iTerm.app and choose "Show Package Contents". Put the localization files within the iTerm directory into the iTerm.app/Contents/Resources, and the files within the iTerm Framework directory into iTerm.Contents/Frameworks/iTerm.framework/Vesrions/A/Resources.
