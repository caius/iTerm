 Navigation:
    Tabs, the toolbar, contextual menus and the bookmarks, all help to aid navigation in iTerm. If you don't use these features, give them a look, because they can be extremely helpful. The tabs support drag and drop enabling you to re-order them within and between windows.
Shortcut keys:
  • Command or Shift-left/right/Up/Down Arrow: Switch to the previous/next tab
  • Command Fn: Switch to n-th session

  • Shift PageUp/PageDown or Shift-Command-Up/Down Arrow: Scroll Up/Down one page
  • Command-Up/Down Arrow: Scroll Up/Down one line

  • Right-Click: Context menu
  • Command Right-Click: Conext menu with open in new window options

  • CTRL-Option-LetterKey: Open a new sesson in tab
  • Shift-CTRL-Option-LetterKey: Open a new session in new window
    • Note: The letter key is specified in Bookmarks. You can assign the same key to different entries. Using this key will load all these entries at once.
Contextual Menu:
  • Has access to your Bookmarks!!
  • Has access to the Configure panel
  • Has access to clear the buffer
  • Selected text can sent to either your web browser, your email program, saved to file or copied.
      Text sent to your mail program should be an email address
      Text sent to your browser should be a URL
Bookmarks:
  • The Bookmarks organizes preset terminal settings.
  • Instead of modifying the configure panel or preference panel, you can save seperate settings for a session
  • For the command you can specify an command, doesn't have to be a shell.
  • The Bookmarks can be accessed from a contextual menu, the Tool Bar or from the menu: Window -> Bookmarks (Command + Shift + A)
Autolaunching script:
    iTerm also supports autolaunching of an Applescript on startup. On startup, iTerm looks for an Applescript called "AutoLaunch.scpt" under ~/Library/Application Support/iTerm. If it is found, the AutoLauch.scpt is launched and executed. This feature can be used to launch multiple sessions in defined windows and tabs when iTerm atarts up.
