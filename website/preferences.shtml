Preference vs Configure panels:
    The Configure panels and the Preference panels are essentially the same thing, with the exception that the Configure panel applies changes to the current window only. The Preference panel changes the default session settings.
Preferences:
  • Anti-Aliasing: Recommended for viewing text easier.
  • Transparency: Is a wonderful thing, but if you have "100%" transparency, when you click in the session window, the mouse click will be sent to the application behind iTerm.
  • Window Size: If you change the size of the session window in the Configure panel it will resize the window to that size regardless of number of session open. However, if you open a new session in a new tab, the window will remain the same size, and not honor the settings of the new session.
  • Shells: The default shells is /bin/tcsh. Other common shells are:
      /bin/sh
      /bin/csh
      /bin/bash
      /bin/zsh.
